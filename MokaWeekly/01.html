Let's Build a Command Line Program in C++

By: Mustafif Khan | 06.24.2021

Before we get into any C++ code, let's first decide what we'll build. We'll start simple then build from there when we have a base, and this idea will be a word counter. Here's what the program will do:

  • Set a file path
  • Read lines in the file
  • Increment by counting spaces in the file
  • Once done, print the result
